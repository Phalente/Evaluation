Qui sommes-nous?

TouchePasAMonStream alias TPMS est une association de streamers confirmé dans le domaine des jeux-vidéos. Cette association à été crée en 2020 suite à un pic d'activité de streamers voulant éclorent dans ce milieu.

Cette montée en puissance dans le domaine n'a pas échappé à notre équipe et, voulant aider ce secteur, nous avons profiter de ce phénomène pour allier streamers en devenir et associations de jeunesse externe afin que les fonds récoltés avec le streaming puissent servir à aider les jeunes défavorisés mais aussi ceux qui ne peuvent pas se payer le matériel nécessaire afin d'exercer.

En plus de les aider, ils pourront suivre certains streamers assez connus dans le milieu afin d'avoir toutes les bases pour réussir dans ce milieu, dans la joie et la bonne humeur.

Entraidons-nous tous ensemble, et faisons de cette association un espoir pour la jeunesse!
